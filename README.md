TrafficCar
=====

Machine Learning Based Network Traffic Analysis and Audit System

Dependencies
-----
```bash
$> sudo apt install build-essential
$> sudo apt install qtbase5-dev qtchooser qt5-qmake qtbase5-dev-tools
$> sudo apt install libpcap-dev
```

Run
-----
```bash
$> cd build
$> chmod +x ./build.sh
$> ./bin/trafficat
```
